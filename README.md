# Batallas RPG en Python

Simulador de batallas RPG en terminal desarrollado en Python como
ejercicio de Programación Orientada a Objetos (POO).

## 🚀 Características actuales

-   Sistema de personajes
-   Clase base `Personaje`
-   Atributos: vida, ataque, defensa, nivel
-   Estructura escalable para RPG por turnos

## 🧠 Objetivo del proyecto

Este proyecto tiene como objetivo practicar:

-   Programación Orientada a Objetos (POO)
-   Herencia
-   Encapsulación
-   Polimorfismo
-   Diseño de arquitectura de juegos

## 🛠️ Tecnologías

-   Python 3.x
-   Terminal / Consola

## 📦 Instalación

Clonar el repositorio:

``` bash
git clone https://github.com/dannygtz93-cyber/EjerciciosPython_BatallasRPG.git
```

Entrar al directorio:

``` bash
cd EjerciciosPython_BatallasRPG
```

Ejecutar:

``` bash
python BatallasRPG.py
```
